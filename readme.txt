This plug-in displays the contents of the child page on a parent page with a tab.

== Description ==
This plug-in displays the contents of the child page on a parent page with a tab.

= Some features =

* The contents of the child page are displayed using tabs in inputting a short code `[inline_navi]` .

* The asynchronous communication in Ajax becomes effective in inputting a short code `[inline_navi ajax='on']` .

* Tabs are displayed under the contents of the child page in inputting a short code `[inline_navi position='bottom']` .

== Installation ==

1. Upload Inline Navi plugin folder you downloaded to the plugin directory.
2. Go to the plugin menu of Admin, and activate "Inline Navi" plugin.
